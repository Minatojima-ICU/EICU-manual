# Claudeで始める

このページは、Claudeを使って最初のバイブコーディングを行うための手順です。  
共通準備がまだの場合は、先に [AIコーディングエージェント入門](./vibe_coding_first_step.md) を読んでください。

Claudeでは、最初はローカルフォルダを直接編集せず、Claudeデスクトップアプリのプロジェクト管理を使って進めます。  
最初の1回目は無料枠または既存プランの範囲で行います。APIキーや従量課金の設定は使いません。

## 1. 向いている人

- Claudeを普段から使っている
- ターミナルを使わずに始めたい
- まずはClaudeデスクトップアプリの画面内で安全に試したい

## 2. 準備するもの

- Claudeアカウント
- インターネット接続
- 30〜60分の作業時間

## 3. Claudeを開く

1. [Claude](https://claude.ai/) を開く
2. Claudeのデスクトップアプリをダウンロードする
3. Claudeデスクトップアプリをインストールする
4. Claudeデスクトップアプリを開く
5. Claudeアカウントがなければ作成する
6. Claudeアカウントがあればログインする
7. 無料枠または既存プランで使える状態になっていることを確認する

このページでは、Claudeデスクトップアプリの通常のチャット画面とプロジェクト機能を使います。  
Claude Code、CLI、IDE連携、ローカルフォルダ編集は使いません。

## 4. プロジェクトを作る

Claudeデスクトップアプリの画面で、練習用のプロジェクトを作ります。

1. 左側のメニューから `Projects` または `プロジェクト` を開く
2. `New project` または `新規プロジェクト` を選ぶ
3. プロジェクト名を `vibe_coding_test` にする
4. 説明欄があれば、`バイブコーディング練習用。患者情報や個人情報は入れない。` と書く
5. 作成した `vibe_coding_test` プロジェクトを開く

この練習では、PC上のフォルダをClaudeに直接操作させません。  
作成物はClaudeの回答として受け取り、内容を確認してから必要に応じて自分で保存します。

## 5. 安全ルールをプロジェクトに伝える

プロジェクトを開いたら、最初に安全ルールを伝えます。  
以下のプロンプトを貼り付けてください。

```prompt
このプロジェクトは、バイブコーディングの練習用です。

以下のルールを守ってください。
- 患者情報や個人情報は扱わない
- APIキーやパスワードは扱わない
- 外部APIや生成AI APIを使うコードは作らない
- 医療内容は学習用として扱い、実臨床で使う正式ツールとは書かない
- 初心者向けに、作業内容を短く説明する
```

Claudeが了解したら、次に進みます。

## 6. 最初のメッセージ

まずは、短いテキストファイルの内容を作ってもらいます。  
以下のプロンプトを貼り付けてください。

```prompt
`ARDS診断基準.txt` というファイルに保存する想定で、内容を作ってください。

内容は、ARDSの診断基準を医療者向けの学習用に短くまとめたものにしてください。
患者情報や個人情報は含めないでください。
内容は初心者にも読みやすいようにしてください。

出力は、ファイル名と本文がわかる形にしてください。
```

## 7. 確認すること

- `ARDS診断基準.txt` に入れる内容が表示されている
- 内容が短くまとまっている
- 患者情報や個人情報が含まれていない
- 実臨床でそのまま使うような断定的な表現になっていない

問題なければ、次に簡単なWebアプリの内容を作ります。

## 8. 次の練習: 簡単なWebアプリを作る

次は、ARDS患者の目標一回換気量を計算する簡単なWebアプリを作ってみます。  
Claudeに、以下のプロンプトを貼り付けてください。

```prompt
## 課題
ARDS患者の目標一回換気量を計算して表示するWebアプリのコードを作ってください。

## 条件
- HTML, CSS, JavaScriptだけで作ってください
- 外部APIや生成AI APIは使わないでください
- 身長(cm)と性別を入力できるようにしてください
- Devineの式でIBWを計算してください
- 6-8 ml/kg IBWの一回換気量の範囲を表示してください
- 医療現場で使う正式ツールではなく、学習用であることを画面に表示してください
- 初心者が保存しやすいように、`index.html` 1ファイルで完結する形にしてください
- 作成後、ブラウザで開く方法も教えてください

## Devineの式
- Male: IBW (kg) = 50 + 2.3 × ((height_cm / 2.54) - 60)
- Female: IBW (kg) = 45.5 + 2.3 × ((height_cm / 2.54) - 60)
```

Claudeがコードを表示したら、まず内容を確認してください。  
このページではClaudeがPC上のファイルを直接作成しないため、必要な場合だけ自分で `index.html` として保存します。

## 9. Webアプリ作成後に確認すること

- `index.html` 1ファイルで完結している
- 入力欄がある
- 計算結果が表示される
- 学習用であることが明記されている
- 外部APIや生成AI APIが使われていない

ブラウザで開く方法がわからない場合は、Claudeに次のように聞いてください。

```prompt
この `index.html` を初心者がブラウザで開く方法を、1ステップずつ教えてください。
```

これで、Claudeのプロジェクト管理を使った初めてのバイブコーディングができました!

## 10. 困ったとき

Claudeで作業内容がわからなくなったら、次のように聞いてください。

```prompt
このプロジェクトでここまで作った内容を一覧にしてください。
それぞれ何のためのものか、初心者向けに説明してください。
```

Claudeの回答はそのまま信用せず、作られた内容を必ず確認してから次の依頼に進んでください。
