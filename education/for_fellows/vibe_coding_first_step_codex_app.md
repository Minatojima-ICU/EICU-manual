# Codex appで始める

このページは、Codex appを使って最初のバイブコーディングを行うための手順です。  
共通準備がまだの場合は、先に [AIコーディングエージェント入門](./vibe_coding_first_step.md) を読んで、`vibe_coding_test` フォルダを作ってください。

最初の1回目は無料枠または既存プランの範囲で行います。APIキーや従量課金の設定は使いません。

## 1. 向いている人

- ChatGPTを普段から使っている
- OpenAIのツールで始めたい
- ターミナルを使わずにデスクトップアプリから試したい

## 2. 準備するもの

- ChatGPTアカウント
- インターネット接続
- `vibe_coding_test` フォルダ
- 30〜60分の作業時間

## 3. Codex appをインストールする

1. [Codex app公式ページ](https://developers.openai.com/codex/app) を開く
2. macOSまたはWindows用のアプリをダウンロードする
3. アプリをインストールする
4. Codex appを開く
5. ChatGPTアカウントでログインする

インストールやログインでうまくいかない場合は、エラーメッセージをそのままコピーして、ChatGPTなどに相談してください。

## 4. 作業フォルダを選ぶ

Codex appで上段の「ファイル」から「フォルダを開く」を選びます。
フォルダを選ぶ画面が出たら、`vibe_coding_test` フォルダを選びます。


*最初は、既存の仕事用フォルダ、研究用フォルダ、個人情報が入っているフォルダは選ばないでください。*

## 5. 作業モードを確認する

Codex appでは、Codexが自分のPC上のフォルダを操作する設定があります。
「承認を求める」「代理で承認」のいずれかになっていることを確認してください。

*最初は「フルアクセス」を避けましょう*

コマンド実行やファイル削除の許可を求められた場合は、内容が理解できるまで許可しないでください。

## 6. 最初のメッセージ

まずは、短いテキストファイルを作ってもらいます。
以下のプロンプトを貼り付けてください。

```prompt
このフォルダに `ARDS診断基準.txt` というファイルを作ってください。

内容は、ARDSの診断基準を医療者向けの学習用に短くまとめたものにしてください。
患者情報や個人情報は含めないでください。
内容は初心者にも読みやすいようにしてください。
```

そして実行してください。codexが動き始めます。
承認を求められたら、その都度承認してください。

## 7. 確認すること

- `vibe_coding_test` フォルダに、`ARDS診断基準.txt` が作られている
- 内容が短くまとまっている
- 患者情報や個人情報が含まれていない
- 余計なファイルが作られていない

問題なければ、次に簡単なWebアプリを作ります。

## 8. 次の練習: 簡単なWebアプリを作る

次は、ARDS患者の目標一回換気量を計算する簡単なWebアプリを作ってみます。  
Codex appに、以下のプロンプトを貼り付けてください。

```prompt
## 課題
ARDS患者の目標一回換気量を計算して表示するWebアプリを作ってください。

## 条件
- HTML, CSS, JavaScriptだけで作ってください
- 外部APIや生成AI APIは使わないでください
- 身長(cm)と性別を入力できるようにしてください
- Devineの式でIBWを計算してください
- 6-8 ml/kg IBWの一回換気量の範囲を表示してください
- 医療現場で使う正式ツールではなく、学習用であることを画面に表示してください
- 作成後、ブラウザで開く方法も教えてください

## Devineの式
- Male: IBW (kg) = 50 + 2.3 × ((height_cm / 2.54) - 60)
- Female: IBW (kg) = 45.5 + 2.3 × ((height_cm / 2.54) - 60)
```

Codex appがファイル作成やコマンド実行の承認を求めてきたら、内容を確認してから承認してください。  
意味がわからない操作や、`vibe_coding_test` 以外のフォルダを変更しようとしている場合は承認しないでください。

## 9. Webアプリ作成後に確認すること

- codex app内でファイルを開くか、フォルダをみて直接htmlファイルを開く
- 入力欄がある
- 計算結果が表示される
- 学習用であることが明記されている
- 外部APIや生成AI APIが使われていない
- `vibe_coding_test` 以外のフォルダが変更されていない

ブラウザで開く方法がわからない場合は、Codex appに次のように聞いてください。

```prompt
作成したWebアプリをブラウザで開く方法を、初心者向けに1ステップずつ教えてください。
```

これで、初めてのバイブコーディングができました！

## 10. 困ったとき

Codex app内で何が起きたかわからない場合は、次のように聞いてください。

```prompt
いま作成・変更したファイルを一覧で教えてください。
それぞれ何のためのファイルか、初心者向けに説明してください。
```

作業を続ける前に、作成されたファイルを必ず確認してください。
