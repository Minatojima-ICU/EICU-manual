# バイブコーディング練習ガイド（初心者向け）

このガイドは、`npm`やターミナルを初めて使う方向けです。  
「最初の1回を安全に完了する」ことをゴールにしています。

## 1. この練習でできるようになること

- ターミナルを開いてコマンドを実行できる
- `Node.js`と`npm`をインストールできる
- CLIでLLM（ChatGPT/Codex, Gemini, Claude Code）を起動できる
- 短い指示文（プロンプト）で簡単な作業を依頼できる

## 2. 事前準備

### 必要なもの

- インターネット接続
- PC（Windows または Mac）
- メールアドレス（各サービス登録用）
- 30〜60分の作業時間

### 注意

- API利用は課金が発生する場合があります。料金ページを先に確認してください。
- 会社PCの場合は、社内ルールに従ってください。

## 3. ターミナルとは

ターミナルは、文字でPCに命令するアプリです。  
最初は「コピーして貼り付けて実行」で問題ありません。

### 開き方

- Mac: `Spotlight`で「ターミナル」を検索して開く
- Windows: `スタート`で「PowerShell」または「Windows Terminal」を検索して開く

### 最初の確認コマンド

```bash
echo "hello"
```

`hello`と表示されたらOKです。

### ターミナルの練習
[ターミナルの練習ページ](./terminal_practice.md) を先に完了してから進んでください。

## 4. Node.js と npm のインストール

CLIツールは`npm`経由で入れることが多いため、先に準備します。

1. Node.js公式サイトで`LTS`版をインストール  
2. ターミナルを再起動  
3. 次を実行して確認

```bash
node -v
npm -v
```

どちらもバージョン番号が出ればOKです。

## 5. CLI用LLMの設定

使いたいツールを1つ選んで進めてください。  
（慣れるまでは1つに絞るのがおすすめ）

### 5-1. ChatGPT / Codex の場合

1. アカウント作成とログイン
2. 利用手順の公式ドキュメントに沿ってCLIをインストール
3. 認証（ログインまたはAPIキー設定）

### 5-2. Gemini の場合

1. Googleアカウントの準備
2. Gemini CLIの公式手順でインストール
3. 認証設定（必要ならAPIキー）

### 5-3. Claude Code の場合

1. Anthropicアカウントの準備
2. Claude Code公式手順でインストール
3. 認証設定（必要ならAPIキー）

## 6. LLMを起動する

インストールしたツールの起動コマンドを実行します。  
例（実際は各ツールの公式コマンドを使用）:

```bash
<tool-command>
```

起動後、入力待ちになればOKです。

## 7. 最初のプロンプト例

まずは短く、1つの作業だけ頼むのがコツです。

- 「このフォルダのファイル一覧を見て、何があるか説明して」
- 「README.mdを初心者向けにやさしい日本語で直して」
- 「次に実行するコマンドを1つだけ示して」

## 8. うまくいかないとき（よくある問題）

### `command not found`

- 原因: インストール未完了、またはターミナル再起動前
- 対処: インストール確認後、ターミナルを閉じて開き直す

### `npm: not found`

- 原因: Node.jsが入っていない
- 対処: Node.js（LTS）を再インストールして`node -v`/`npm -v`を確認

### 権限エラー（permission denied）

- 原因: 実行権限や保存先の制限
- 対処: 管理者権限のターミナルで再実行、または保存先フォルダを変更

## 9. 安全ルール（重要）

- 意味がわからないコマンドは実行しない
- `rm -rf`など削除系コマンドは特に慎重に扱う
- APIキーやパスワードをチャットに貼らない
- エラーが出たら、エラーメッセージをそのままLLMに見せて相談する


## 10. 初めてのバイブコーディング
次のプロンプトをLLMに渡して、簡単なアプリを作ってみましょう。
```prompt.txt
## 課題
ARDS患者の目標一回換気量を計算して表示するwebアプリを作りたい。

## 内容
以下の条件にあうようにwebアプリを作成してください。

## 条件
- javascriptで記述してください
- `身長(cm)`と`性別 (m or f)`を入力したら、6-8 ml/kg (IBW)の値に合致するように、適切な一回換気量の範囲を表示してください。
- 以下のDevineの式を使ってください。
  - Male: IBW (kg) = 50 + 2.3 × ((height_cm / 2.54) − 60)
  - Female: IBW (kg) = 45.5 + 2.3 × ((height_cm / 2.54) − 60)
```


## 11. 次のステップ

1. 1日1回、短い作業をLLMに依頼する
2. プロンプトを作った時に、次のようにLLMに聞いてより良いプロンプトに変更する。
```
以下のプロンプトを、適切な構造化を実施し、洗練させて、"~~~"で括って、私にコードブロックで提示してください。
```
3. 成功したプロンプトをメモして再利用する
